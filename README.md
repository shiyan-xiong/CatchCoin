# 🪙 接金币游戏 (CatchCoin)

一个像素风格的接金币网页游戏，使用 HTML + CSS + JavaScript 实现。

## 🎮 游戏介绍

在游戏中，你需要控制盘子接住从天而降的金币，同时避开石头。大金币价值更高，但石头会扣分！

## ✨ 游戏特色

- 🎨 **像素风格**：经典的像素艺术风格，怀旧感十足
- 🎯 **三种物品**：
  - 🪙 大金币：+2分
  - 🪙 小金币：+1分
  - 🪨 石头：-1分
- ⚡ **动态速度**：游戏进行中速度会逐渐增加，挑战性不断提升
- 🏆 **最高分记录**：自动保存最高分到本地存储
- ⚙️ **个性化设置**：可自定义背景颜色

## 🎯 游戏规则

1. 点击"开始"按钮开始游戏
2. 使用鼠标移动或键盘（A/D 或 ←/→）控制盘子左右移动
3. 接住金币得分，避开石头
4. 游戏速度会随时间逐渐增加
5. 点击"暂停"可以暂停游戏
6. 点击"重玩"可以重新开始游戏

## 🕹️ 操作方式

### 鼠标控制
- 在游戏区域内移动鼠标，盘子会跟随鼠标位置

### 键盘控制
- **A** 或 **←**：向左移动
- **D** 或 **→**：向右移动

## 🎨 界面说明

### 游戏信息栏
- **左上角**：当前得分
- **中间**：历史最高分
- **右上角**：当前游戏速度

### 控制按钮
- **开始**：开始新游戏或继续已暂停的游戏
- **暂停**：暂停/继续游戏
- **重玩**：重新开始游戏

**按钮状态说明**：
- 游戏未开始时：暂停和重玩按钮为灰色（不可用）
- 游戏进行中：开始按钮为灰色（不可用）
- 游戏暂停时：所有按钮可用

### 设置面板
点击右上角的 ⚙️ 图标打开设置面板，可以：
- 调整游戏背景颜色

## 🚀 快速开始

1. 下载或克隆项目文件
2. 在浏览器中打开 `index.html` 文件
3. 点击"开始"按钮即可开始游戏

## 🎮 游戏技巧

1. **保持移动**：盘子移动速度有限，提前预判物品掉落位置
2. **优先接大金币**：大金币得分更高，但也要注意避开石头
3. **速度管理**：随着速度增加，反应要更快
4. **合理暂停**：需要休息时可以暂停游戏
